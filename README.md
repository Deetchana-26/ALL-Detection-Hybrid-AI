# ALL-Detection-Hybrid-AI
This project presents a hybrid deep learning framework for Acute Lymphoblastic Leukemia (ALL) detection using blood smear images.

The system combines:
- ResNet50
- Vision Transformer (ViT)
- Harris Hawk Optimization (HHO)
- Particle Swarm Optimization (PSO)
- XGBoost, SVM, and MLP classifiers
- Explainable AI (SHAP & Attention Maps)

The framework improves classification accuracy, robustness, and interpretability for AI-assisted leukemia diagnosis.
